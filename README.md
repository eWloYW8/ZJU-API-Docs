# ZJU API Docs

非官方的浙江大学 API 文档项目，收录了「学在浙大」「智云课堂」「统一身份认证」等平台的接口信息，旨在为二次开发、学习研究提供参考。

**本项目中的 API 均为校内平台前端的公开接口，使用时与浏览器访问校内平台并无差异，不含任何内部 API 和提权漏洞**
